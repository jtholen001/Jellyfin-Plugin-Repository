# Jellyfin-Plugin-Repository


<h1 align="center">Jellyfin-Plugin-Repository</h1>
<p align="center">A repository for my Jellyfin plugins. </p>
<h3 align="center">Part of the <a href="https://jellyfin.org">Jellyfin Project</a></h3>


## Use Guide

1. In Jellyfin, go to dashboard -> plugins -> Repositories -> add and paste this link ```https://raw.githubusercontent.com/jtholen001/Jellyfin-Plugin-Repository/main/manifest.json```
2. Go to Catalog and search for the plugin you want to install
3. Click on it and install
4. Restart Jellyfin
